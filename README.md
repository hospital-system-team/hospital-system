# Hospital System

A modern web-based hospital management system built with HTML, CSS, and JavaScript, integrated with Supabase for backend services.

## Features

- Modern and responsive user interface
- Hospital management functionality
- Supabase integration for data management
- Clean and maintainable code structure

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Supabase
- Yarn package manager

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/hospital-system-team/hospital-system.git
   cd hospital-system
   ```

2. Install dependencies:
   ```bash
   yarn install
   ```

3. Set up your Supabase configuration in `supabase.js`

4. Open `index.html` in your browser

## Project Structure

- `index.html` - Main HTML file
- `style.css` - Stylesheet
- `supabase.js` - Supabase configuration and functions
- `yarn.lock` - Yarn lock file
